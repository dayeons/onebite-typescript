- 제네릭은 타입을 변수의 형태로 받아 문맥에 따라 다른 결과 타입을 만든다. 기존 타입을 상황에 맞게 변환하는 타입 조작의 한 형태
- `keyof`는 객체 타입의 속성 이름들을 모두 가져와 문자열 리터럴 유니온 타입으로 만든다. 이를 통해 안전하게 객체 속성에 접근하는 타입을 정의
- 맵드 타입은 객체 타입의 각 속성을 순회하며 새로운 속성 타입을 정의해서 전혀 다른 형태의 객체 타입을 효율적으로 생성해주는 기능
- 템플릿 리터럴 타입은 자바스크립트의 템플릿 리터럴 구문처럼 백틱을 사용해 여러 타입들을 조합한 새로운 문자열 리터럴 유니온 타입을 자동을 생성
